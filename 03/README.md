# Coding Challenge - 03

Problem: Given integer n find  out if it can be reflected

Legend:
* | - "mirror"

Example:
* 12|3     ❌ not reflective
* 12|21    ✅ reflective
* 123|123  ❌ not reflective
* 123|321  ✅ reflective
